Eclipse Build process


Using multiple versions of Java with Eclipse.

To add a new version of JVM to Eclipse : Click on Add →

Click Next → we need select our JDK → here in this case, we have selected JDK 1.8

Click Finish.

Here, you select jdk1.8 and click on Apply → checked JRE is added to the build path of newly created java projects.
Also change compiler compliance settings.



How to change Java buildpath for existing project :

Alternatively, click on Project → Properties

Go under Libraries →


Click on Apply.

After this, you also need to set Compiler compliance as well.


Importing source files into Eclipse project

This way, we don't even need to refresh the project, the source files will automatically be added to the project.


Source code Navigation


type exploration — class / interface

Ctrl + Shift + T


Outline of a class — like methods, fields of a class

Ctrl + O (or) right click in class definition --> Show in Breadcrumb

To navigate to method definition / method implementation — click on method name and hit F3

To know what all methods calling this function — click on method name and hit Ctrl + Shift + G

To know method calling hierarchy — Ctrl + Alt + H

Search text within the files — Ctrl + H

Go to particular line — Ctrl + L

Summary :

Ctrl + Shift + T ~ Searching for a type

Ctrl + O ~ Quick outline

Ctrl + T ~ Quick type hierarchy

F3 ~ Navigate to a method

Ctrl + Shift + G ~ Method invocations

Ctrl + ALT + H ~ Method call hierarchy

Ctrl + H ~ Search console

Ctrl + Spacebar ~ Keyword (e.g., foreach) followed by this shortcut will auto-insert a template
